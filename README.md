This project is an automated car parking detection system built using Python and OpenCV.
It detects available parking slots in real-time using image processing techniques.

## Features
- Real-time car detection using OpenCV
- Slot marking and counting
- Easy-to-customize interface

## How to Run
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python main.py`
